## Video URL Extraction Guide

### Finding YouTube Video ID
1. Go to video page
2. Look at URL: `https://www.youtube.com/watch?v=0IAPZzGSbME`
3. Video ID is after `v=`: **0IAPZzGSbME**

### Creating Embed URL
Replace `watch?v=` with `embed/`:
- Original: `https://www.youtube.com/watch?v=0IAPZzGSbME`
- Embed: `https://www.youtube.com/embed/0IAPZzGSbME`

### Getting Thumbnail
Use video ID with thumbnail URL:
- Format: `https://i.ytimg.com/vi/[VIDEO_ID]/maxresdefault.jpg`
- Example: `https://i.ytimg.com/vi/0IAPZzGSbME/maxresdefault.jpg`

### Finding Duration
1. Click on video
2. Look at video description
3. Duration is shown in video player
4. Convert to HH:MM:SS format

If you want to add many videos at once, you can use this SQL query template:

```sql
INSERT INTO public.videos 
(title, instructor, description, thumbnail_url, video_url, duration, duration_seconds, category, tags, published_date, is_published) 
VALUES 
(
  'Video Title',
  'Instructor Name',
  'Video description...',
  'https://i.ytimg.com/vi/.../maxresdefault.jpg',
  'https://www.youtube.com/embed/...',
  'HH:MM:SS',
  [duration_in_seconds],
  'category',
  ARRAY['tag1', 'tag2', 'tag3'],
  'YYYY-MM-DD',
  TRUE
);
```
